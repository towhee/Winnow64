RAW DECODE / DEVELOP - TASK LIST
================================================================================
Goal: decode, render and edit raw files natively in Winnow (no exiftool/libraw at
runtime), each format validated byte-identical to libraw. All gated by G::useRaw.

PERSISTED 2026-06-24. The original list lived in the "Raw decoder and editor"
conversation and was never saved to a file; this restores it from the project
record + current code. Statuses verified against the tree on this date.

Pipeline (ImageFormats/Raw/rawformat.cpp Decode):
  UnpackCfa (per-format) -> SubtractBlack -> Demosaic -> RawColor -> Develop -> OutputTransform

15   [done]      Implement the real Develop ops on the scene-linear buffer.
                 DONE (2026-06-25): all ten Basic sliders live (Develop/develop.{h,cpp}).
                 POINT ops (WB temp/tint, exposure, contrast, tone regions
                 highlights/shadows/whites/blacks) = a single FUSED, QtConcurrent-parallel
                 per-pixel pass: WB+exposure fold into a per-channel linear gain; contrast +
                 tone regions bake into a 1-D perceptual (~gamma-2.2) LUT (PointCoeffs::toneLut)

                 after the point pass: Texture (#6, mid-freq luma local contrast) and Dehaze

                 the proxy preview matches full-res. See Documentation.txt "DEVELOP / IMAGE

                 radiance. ~120-150 lines, heavier on the full-res settle render, more
                 parameters to tune; physically removes real haze (better on hazy
                 landscapes). Would slot into Develop::Dehaze behind the same slider.

23   [DONE 2026-06-28] Wire RGB and HSL sliders under the Color Header. All six are POINT

                 block after the tone curve -- hue = Rodrigues rotation about the (1,1,1) neutral
                 axis, saturation = chroma scale about Rec.709 luma, luminance = uniform gain.
                 EditParams fields + JSON persistence (editstack.h) + UI wiring
                 (applyKeyToParams, populateSlidersFromStack) all live. See
                 Documentation.txt "DEVELOP / IMAGE EDIT".

24   [done]      Vignette (Effects). Radial corner-exposure falloff about the
                 image centre (Develop::Vignette, spatial op #8): mask = pow(rn,
                 k), rn the elliptical radius (0 centre, 1 corner), k from the
                 Feather slider; corner gain = 2^Exposure. Two sliders
                 (vignetteExposure EV + vignetteFeather). Off-centre / shaped
                 vignettes use radial masks.

25   [done]      Grain (Effects). Monochromatic film grain added to perceptual

                 sliders (grainAmount/grainSize/grainRoughness, 0..100 -> 0..1).
                 EditParams + JSON + UI wiring live.

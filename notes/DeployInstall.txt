For Win:
-------------------------------
Win Deployment:
CD "C:\program files\winnow"
C:\Qt\5.10.0\msvc2017_64\bin\windeployqt.exe "D:\My Projects\Winnow Project\build-winnow64-Desktop_Qt_5_10_0_MSVC2017_64bit-Release\release\Winnow.exe" --dir "C:\Program Files\Winnow"

Win Install:
Repository
CD D:\My Projects\Winnow Project\WinnowInstallOnline
C:\Qt\Tools\QtInstallerFramework\2.0\bin\repogen.exe -p packages repository
Use winscp.exe to copy repository to DigitalOcean web server
Installer
CD D:\My Projects\Winnow Project\WinnowInstall
C:\Qt\Tools\QtInstallerFramework\2.0\bin\binarycreator.exe --offline-only -c config/config.xml -p packages winnowinstalleroffline.exe
C:\Qt\Tools\QtInstallerFramework\2.0\bin\binarycreator.exe --online-only -c config/config.xml -p packages winnowinstalleronline.exe

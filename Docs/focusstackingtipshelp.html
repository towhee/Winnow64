Focus Stacking Tips

Focus stacking combines a series of images shot at different focus distances into a single image with greater depth of field. Winnow can process one stack at a time or a batch of several stacks in one operation.

Remote vs Local mode

Winnow can run the focus stacking computation in two modes:

  • Local mode runs the stacking engine on this computer. All processing uses your machine's CPU and memory. Use this when you are working offline or prefer to keep everything on your own hardware. Performance depends on the number of images, their resolution and the amount of available memory.
  • Remote mode sends the images to a dedicated stacking server which performs the computation and returns the finished result. This offloads the heavy processing from your machine, which is useful for large stacks or when working on a less powerful computer. Remote mode requires a network connection to the server.

Choose the mode that best matches your hardware and connectivity. Local mode keeps the data on your machine; remote mode trades a network transfer for faster or more capable processing.

Batch selection of multiple stacks

You do not have to stack one group of images at a time. Select the images for several stacks and Winnow will detect the separate stacks and process them in sequence as a batch.

  • Select all of the images that belong to the stacks you want to process.
  • Winnow groups the selection into individual stacks and queues each one.
  • Each stack is processed in turn and its result is produced before the next stack begins.

Batch processing lets you set up many stacks and leave Winnow to work through them unattended.

Status progress bar

While a stack or batch is being processed, a progress bar in the status bar shows how far along the operation is.

  • The progress bar advances as each image in the current stack is processed.
  • When processing a batch, it reflects progress through the entire queue of stacks.
  • The status bar text reports the current stage so you can see what Winnow is doing at each step.

You can continue to monitor progress here until the stacking operation completes.

Find duplicate images and videos

You have two groups of images in folders: the candidate images and the target images. We want to determine if any of the candidate images already exist in the target images group.

Step 1: Pick the candidates

The candidate images are the selected images in the currently selected folder(s) in Winnow, shown in the right side of the Find Duplicate Images dialog. Select the images you want to check before opening the dialog; if nothing is selected, all of the filtered images are used as candidates. If there are not any images, close the dialog, select the folder with the candidate images, select the images to check (filter if necessary) and run the Find Duplicate Images dialog again.

Step 2: Pick the target group

Using finder or explorer, drag folders into the include list located at the top of the dialog. If a folder has a hierarchy of subfolders you also want to include, then also check include subfolders. If you want to exclude any folders in the hierarchy then drag the exclude folders to the exclude list. Excluded folders do not include subfolders.

Step 3: Set the search criteria

The criteria for duplication is either file metadata or image pixels. It is toggled by checking same pixels. If same pixels is checked you can set the pixel delta threshold (see adjacent ? for more info). A perfect match = 0 delta. If same pixels is not checked, then check the metadata criteria you wish to use to find duplicates (file name, file type, creation date, aspect, duration). Duration for images is always 0, so it is only relevant for videos and harmless to include for images.

Step 4: Search for duplicates

Click the Find Duplicates button. When the search is completed, duplicate images will be checked in the candidate images list on the right.

Step 5: Confirm duplicates

The candidate Images list (right side of dialog) shows the number of duplicates found, and if the criteria = same pixels, the pixel delta is also shown. If you select an image in the candidate Images list, the compare images box will show the candidate and matching target image(s). You can use this to confirm duplicates. If it is not a duplicate then uncheck the image in the Candidate Images list.

Use the Report button to see a detailed comparison of the candidate and matching target images, including the metadata or pixel deltas. If you want to include all the non-matches, press alt/option and Report.

If you are getting false matches, this might be due to identical creation times for multiple images taken in rapid succession, and the camera did not include millisecond time information. In this case you should use the same pixels criteria.

Step 6: Do something with the duplicates

Click the Done button. In Winnow, the filter panel will now contain a Duplicates found category. If you filter on the true items you can then delete, copy or move the images.

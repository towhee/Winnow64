Embellish Overview

The Embellish panel provides the tools to build embellish templates, showing a preview of the template as you build it. Each template is a set of instructions. Also see videos on the Winnow web site.

Template categories

General: Define the size and light direction.

Borders : Add, remove and edit borders.

Text: Add, remove and edit text objects.

Graphics: Add, remove graphic objects.

Styles: Define styles that can be applied to any General, Border, Text or Graphic.

Export: Define where and how to create a new image file from the template, showing the embellished image.

Container and Coordinates

Each template has a container that defines the size of the embellished image. The coordinate system for all objects in the template is relative to the container, with (0, 0) at the top left and (1, 1) at the bottom right. This makes it easy to create templates that work with any image size.

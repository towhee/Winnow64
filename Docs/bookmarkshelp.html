How bookmarks work

The Bookmarks panel keeps a list of folders you frequently visit so you can return to them without navigating the folder tree each time.

Add a bookmark

Drag a folder from the Folders panel (or from Finder/Explorer) onto the Bookmarks panel. The folder is added to the list and stays there between sessions.

Open a bookmark

Click a bookmark to select that folder. The image list and thumbnail view update to show the eligible images in that folder.

Remove a bookmark

Right-click a bookmark and choose Remove Bookmark. The folder itself is not affected — only the shortcut is removed from the panel.

Image counts

If image counts are enabled in Preferences, each bookmark shows the number of eligible images in that folder. Use the refresh button in the title bar to recompute the counts.

Drag and drop

You can drop images onto a bookmark to copy or move them into that folder. Hold the modifier keys to choose the operation:

  • No modifier — copy
  • Shift — move

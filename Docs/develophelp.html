Develop Module

Overview

The develop module edits the image, changing the light, color and effects. The changes are parametric, like a recipe. The image file is not changed, instead the develop recipe is saved in a sidecar file (same name as image file with the .xmp extension).

Layers

The base layer is the original image. All edits are applied to the entire image. Raw image noise reduction is applied to the base image.

Additional layers each contain a layer mask. The layer mask is created and modified using mask tools (linear gradient, brush ...). Each mask tool can add to or subtract from the layer mask. While a mask is active, the edits to Basic, Color and Effects are localized to the layer mask.

Raw Files

In the base layer you can choose to source: either the raw data or the embedded preview image all cameras save in the raw files. If you select 'Raw' then you can select how to demosaic (convert raw data into pixels). If you choose Winnow to demosaic, then you can also run noise reduction on the raw data, which is much better than running noise reduction later in the develop pipeline when the pixels are already 'baked'.

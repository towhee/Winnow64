How the metadata panel works

The Metadata panel shows information about the currently selected image, organised into four categories.

File

Path, filename, size, created/modified dates, dimensions and megapixels — data that comes from the file system and the image header.

Image

Camera, lens and exposure information read from the EXIF block: model, lens, shutter speed, aperture, ISO, exposure compensation and focal length.

Tags

Editable IPTC/XMP fields: title, creator, copyright, email, URL, GPS coordinates and keywords. Double-click a value to edit it. Changes are written back to the file (or to a sidecar) when you save.

Edit

Items whose name is marked with an asterisk (*) can be edited. Double-click the value to open an editor; press Enter to commit or Esc to cancel. Items without an asterisk are read-only.

Status

Runtime information about the current session: position in the data model, zoom factor, selection and pick state, cache usage, free memory and the active monitor.

Choose what to show

Use the gear button in the title bar to open Preferences and choose which rows are visible. Hiding rows you do not use keeps the panel compact and easier to scan.

Copy a value

Right-click a row and choose Copy to put the value on the clipboard.

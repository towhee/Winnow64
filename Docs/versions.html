Winnow Version History


Version 0.9.7.0.2 2019-02-01
  • Fixed a glitch in boosting scroll performance in the folders panel.
Version 0.9.7.0.2 2019-02-01
  • Added milliseconds token (available in ingest and shooting info).
  • Hour, minute and second tokens were not working.
  • Refresh context in bookmarks (favs) updates image count in folders.
  • Folders panel scrolling speed much faster.
  • Fixed issue where mouse clicking on the image count in Favs the folder would not load.
  • You can now add favourite folders to the favs panel by dragging from Windows Explorer or Mac Finder. Previously you could only drag from the folders panel.
Version 0.9.7.0.1 2019-01-31
  • Corrected some errant behaviour in the thumb, grid and table views when mouse click or double mouse click.
Version 0.9.7.0.0 2019-01-30
  • Added the ability to copy to a backup folder as well as the primary ingest location. It is also now possible to open the primary ingest location in Winnow after the ingest is completed.
Version 0.9.6.6.2 2019-01-25
  • Corrected faulty selection behavior.
  • Fixed issue that could cause app crash in Compare mode.
  • Write a metadata sidecar file when open external program in case it can read sidecars.
Version 0.9.6.6.1 2019-01-23
  • Jump to recent ingest history selected folder.
Version 0.9.6.6 2019-01-23
  • Reworked the external program manager to prevent resorting, reorder the apps to preferred shortcuts and cleaned up the user interface.
Version 0.9.6.5 2019-01-22
  • Wider scrollbars and separators.
  • Added another shortcut to pick (P).
  • Added scrolling shortcuts.
Version 0.9.6.4 2019-01-19
  • Added optional automatic check for updates. This can be turned on/off in preferences.
  • Corrected problem where the USB drive would not be ejected when executed from the main menu.
  • Corrected issues when change modes and select invalid folders.
Version 0.9.6.3 2019-01-17
  • Compare images issues fixed.
  • Dragging app window off screen would cause crash.

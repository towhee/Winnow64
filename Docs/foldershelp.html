How folders work

Mouse clicking selects a folder. All selected folders' eligible images are displayed. You can change the selection behavior with the mouse modifier keys:

No modifier

The current selection of folders is cleared and the clicked folder is selected.

Ctrl/Cmd modifier

The folder selection is toggled. If the folder was selected then it will be unselected. If it was unselected then it will be selected. If only one folder is selected then it cannot be unselected.

Shift modifier

All folders from the prior selection to the current clicked folder will be selected. Toggle is ignored.

Alt/Opt modifier

The folder and all subfolders are recursively selected. Combine with Ctrl/Cmd to toggle selection. If only the Alt/Opt modifier is used, then any prior selection is cleared. If there are too many subfolders then the root folder will not expand, but all the subfolders will be loaded and the root folder text will change to orange.
